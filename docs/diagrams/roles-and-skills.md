## Roles and responsibilities
Backend Engineer

    Build and maintain APIs and services

    Ensure system performance and scalability

DevOps Engineer

    Manage CI/CD pipelines

    Maintain cloud infrastructure and monitoring

Data Engineer

    Design data pipelines

    Manage databases and ensure data quality

Security Engineer

    Implement security systems

    Conduct security audits and compliance

SRE

    Maintain system reliability and uptime

    Handle incidents and capacity planning
